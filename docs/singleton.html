点击选择一种实现，然后多次调用 getInstance() 验证是否每次返回同一个对象。

懒汉式 (Lazy)

第一次调用时才创建实例。非线程安全，多线程可能创建多个实例。

懒加载 ✓ 线程不安全 ✗

饿汉式 (Eager)

类加载时就创建实例。线程安全，但会提前占用资源。

非懒加载 ✗ 线程安全 ✓

双重检查锁 (DCL)

加 volatile + synchronized 双重判断，兼顾懒加载和线程安全。

懒加载 ✓ 线程安全 ✓

IoDH 静态内部类

利用类加载机制保证线程安全，性能最优，推荐使用。

懒加载 ✓ 线程安全 ✓ 性能最高 ✓
